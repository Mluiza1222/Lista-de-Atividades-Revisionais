# Lista de Atividades Revisionais

## Sobre o Projeto

Este projeto consiste na resolução de uma lista de exercícios com foco no aprendizado e prática de HTML e CSS.

As atividades foram desenvolvidas com o objetivo de explorar desde a estrutura básica de páginas web até a criação de elementos mais estilizados e interativos.

---

## Objetivo

O projeto tem como principais objetivos:

Aplicar conhecimentos de HTML na construção de páginas
Utilizar CSS para estilização e organização visual
Compreender a separação entre estrutura e estilo
Praticar a criação de diferentes elementos de uma página web
Desenvolver lógica e organização no desenvolvimento front-end

---

## Estrutura dos Exercícios

A lista é composta por diversos exercícios que abordam:

Criação de páginas com estrutura básica
Estilização de textos, cores e fontes
Listas ordenadas e não ordenadas
Links externos
Tabelas com dados organizados
Formulários com diferentes tipos de entrada
Inserção e estilização de imagens
Criação de botões personalizados
Uso de divs para construção de layout
Aplicação de bordas e fundos personalizados
Gradientes e estilização de página completa
Efeitos visuais com hover, transições e animações
Layout responsivo com Flexbox
Menu de navegação
Elementos interativos simples (como modais)
Rodapé fixo

---
 
## Organização dos Arquivos

Os exercícios foram desenvolvidos no VSCode e organizados em arquivos separados, seguindo o padrão:

* exercicio1.html
* exercicio1.css
* exercicio2.html
* exercicio2.css
...
exercício20.html

Cada exercício possui sua própria estrutura HTML e, quando necessário, um arquivo CSS externo.

---

## Tecnologias Utilizadas

* **HTML5** → estrutura das páginas
* **CSS3** → estilização e layout

---

## Considerações Finais

Este projeto foi desenvolvido com foco na prática dos fundamentos do desenvolvimento web.

Através dos exercícios, foi possível reforçar conceitos importantes e desenvolver maior familiaridade com a criação e estilização de páginas utilizando HTML e CSS.
